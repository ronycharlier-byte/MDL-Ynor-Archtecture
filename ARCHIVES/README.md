# ARCHIVES

This layer holds relayed memory, mirrors, releases, logs, exports, audits, and old versions.

ARCHIVES are not first-rank doctrine.
They may support traceability, but they do not govern retrieval.

If a document has a competing canonical source, it is archive-bound.
If a document is only historical evidence, it remains here and does not enter primary retrieval.
