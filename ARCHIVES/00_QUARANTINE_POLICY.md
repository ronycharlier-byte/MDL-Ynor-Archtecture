# Quarantine policy

Archive-bound material:
- older versions
- mirrors
- releases
- changelogs
- logs
- auto-audits
- duplicates
- exports
- prompt wrappers

Rule:
- archive material stays outside the primary retriever.
- if a historical file can be confused with doctrine, it must stay in ARCHIVES with an explicit pointer to the canonical source.
