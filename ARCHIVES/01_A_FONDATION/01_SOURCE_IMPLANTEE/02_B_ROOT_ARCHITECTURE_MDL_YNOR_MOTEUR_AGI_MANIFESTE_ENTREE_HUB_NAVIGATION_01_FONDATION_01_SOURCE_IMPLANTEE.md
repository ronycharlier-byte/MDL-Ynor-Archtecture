## Structure du Projet

### 1. Points d'Entrée & Gestion
- **`LANCER_MOTEUR_AGI.bat`** : Le script maître pour démarrer l'audit AGI et le moteur de raisonnement.
- **`MDL_Ynor_Framework/`** : Le cœur technologique contenant tout le code source, la documentation technique et les couches théoriques.
- **`_RELEASES/`** : Dossier contenant les versions archivées, les snapshots "Phase III" et les exports compressés.
- **`_SUBMISSIONS/`** : Zone de préparation pour les soumissions académiques, incluant les lettres d'accompagnement et les listes de contrôle.
