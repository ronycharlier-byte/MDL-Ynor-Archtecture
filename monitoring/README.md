# MONITORING

This layer holds phase 4 governance, anti-drift policy, event tracking, health reports, historical health, dashboards, and quarantine monitors.

MONITORING is operational.
It watches the system, but it does not define doctrine.

If a document is about health, drift, events, or monitoring behavior, it belongs here.
