# MDL Ynor

Private documentary and runtime corpus.
No environment data or API keys are stored here.

## Scope

- Canonical doctrine lives in `CANON`.
- Chunkable support lives in `RAG_SUPPORT`.
- Historical material lives in `ARCHIVES`.
- Code, scripts, configs, dashboards and generated artefacts live in `TECHNIQUE`.
- Audits and benchmarks live in `AUDIT`.
- Active observability lives in `MONITORING`.
- Ambiguous material lives in `INBOX_QUARANTAINE`.
