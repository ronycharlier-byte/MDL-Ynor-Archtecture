# Mu and survival

Query anchors:
- Definis mu et la loi de survie documentaire.
- Quel est le seuil de survie documentaire ?

Question:
- What is the documentary survival margin?

Answer:
- `mu = alpha - beta - kappa`
- `mu <= 0` means documentary collapse.
- `0 < mu < 3` means fragile.
- `3 <= mu < 5` means serious RAG is possible.
- `mu >= 5` means production window.

Definition:
- `mu = alpha - beta - kappa`

Mathematical Form:
- State $S(t) \in \mathcal{H}$ in a real Hilbert space.
- Dynamics: $\dot S(t) = \mathcal{E}(S(t)) - \mathcal{D}(S(t)) + \mathcal{M}(S_t) + w(t)$
- Where $\mathcal{E}$ is amplification, $\mathcal{D}$ is dissipation, $\mathcal{M}$ is memory, $w$ is perturbation.

Operational reading:
- `mu <= 0` -> documentary collapse
- `0 < mu < 3` -> fragile
- `3 <= mu < 5` -> serious RAG possible
- `mu >= 5` -> production window

Use `mu` as a survival margin, not as a decorative metric.
