# Canonical identifiers

Query anchors:
- Que signifie canonical_id ?
- Explique le principe 1 concept = 1 source canonique.

Question:
- What does `canonical_id` mean?

Answer:
- `canonical_id` is the stable authority identifier for one canonical source.
- `citation_id` is the stable citation handle for one subsection or chunk.
- `concept_id` is the semantic concept label.
- One concept has one canonical source.

Definitions:
- `canonical_id` = stable authority identifier for one canonical source
- `citation_id` = stable citation handle for one subsection or chunk
- `concept_id` = semantic concept label
- `chunk` = local chunk number inside a document

Rules:
- one concept = one canonical source
- one canonical source = one authority line
- one citation_id = one resolvable citation target
