# Operator D

Query anchors:
- Quel est l operateur D et son role ?

Question:
- What is operator D and what does it do?

Answer:
- `D(S) = proj_{SafeDomain}(S)`
- it removes unsafe or non-resolvable states from the working set
- it corrects, it does not invent

Corrective operator:
- `D(S) = proj_{SafeDomain}(S)`

Role:
- remove unsafe or non-resolvable state from the working set
- keep the system inside the safe domain
- correct, not invent

Do not confuse D with a generative operator.
