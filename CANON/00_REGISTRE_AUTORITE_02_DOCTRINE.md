# Shard de doctrine du registre

Purpose:
- Router les questions doctrinales vers la source canonique la plus petite.
- Refuser la reponse quand une fiche dediee existe deja.

Question:
- Quel est le role du registre d autorite ?

Answer:
- The registry routes doctrinal questions to the smallest canonical sheet.
- It decomposes the pivot into finer units.
- It does not replace doctrine.
- If a dedicated sheet exists, the registry must point to it.
